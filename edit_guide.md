# 도원시스템 홈페이지 편집 및 메뉴 추가 가이드 (Edit Guide)

이 가이드는 홈페이지의 **메뉴 추가(제품소개, 시공후기)** 방법과 **사진 직접 교체** 방법을 이해하기 쉽게 설명합니다.

---

## 📂 1. 사진 직접 교체하는 방법 (비주얼 편집기 활용 - 권장)

기존의 코드 수정이나 폴더 복사 없이, 브라우저 화면상에서 **마우스 클릭만으로 내 컴퓨터(또는 휴대폰)의 사진을 웹사이트에 바로 삽입**할 수 있습니다.

### 💡 편집기 활용 방법:
1. 수정할 페이지(`index.html`, `products.html`, `about.html` 중 하나)를 웹 브라우저(크롬 등)로 엽니다.
2. 화면 우측 하단의 **[🔧 편집 모드 시작]** 버튼을 클릭합니다.
3. 바꾸고 싶은 사진(히어로 배경 또는 제품 사진) 위에 마우스를 올리면 나타나는 **[🖼️ 사진 변경]** 버튼을 누릅니다.
4. 팝업창에서 **[📁 내 컴퓨터에서 사진 선택 (자동 업로드)]**을 클릭하여 교체하고 싶은 사진 파일을 선택합니다.
   *(경로 입력을 원하시는 경우 [🔗 이미지 파일 경로 직접 입력]을 사용하여 `images/lg1.jpg`와 같이 입력할 수도 있습니다.)*
5. 화면에 사진이 즉시 반영된 것을 확인한 후, 우측 하단의 **[💾 변경사항 저장 (다운로드)]** 버튼을 클릭합니다.
6. 다운로드된 새 HTML 파일로 기존 프로젝트 폴더 안의 기존 파일을 덮어쓰기(교체)하시면 저장이 완벽하게 완료됩니다.

### 🖼️ 권장 사진 규격 정보
| 이미지 위치 | 권장 비율 / 크기 | 추천 내용 |
| :--- | :--- | :--- |
| **메인/서브 히어로 배경** | `16:9` 비율 (`1920x1080` 이상) | 시스템 에어컨이 조화롭게 설치된 거실/사무실 인테리어 또는 깔끔한 천장 공조 사진 |
| **제품 카탈로그 카드** | `4:3` 비율 (`800x600` 권장) | 브랜드별 천장형/스탠드형 실내기 컷 및 순정 기판/압축기 세부 컷 |
| **회사소개 CEO 인사말 옆** | 세로형 또는 `1:1` (`800x1000` 권장) | 대표이사 프로필 사진 또는 신뢰감을 주는 엔지니어링 작업 실측 모습 |

## 🛠️ 2. 제품소개 및 시공후기 메뉴 추가 방법
이미 `product-intro.html` (제품소개)과 `reviews.html` (시공후기) 파일이 프리미엄 레이아웃으로 완벽히 설계되어 저장되어 있습니다.
이 메뉴들을 홈페이지 상단과 하단에 노출시키려면 아래 단계만 따라 하시면 됩니다.

### Step 1: 상단 헤더 메뉴 주석 풀기
각 HTML 파일(`index.html`, `about.html`, `products.html`, `product-intro.html`, `reviews.html`)의 코드 **상단 헤더(Header)** 영역으로 이동합니다.
아래와 같이 작성된 코드 부분을 찾습니다.

```html
<!-- <li><a href="product-intro.html" class="nav-link" id="nav_prod_intro">제품소개</a></li> -->
<!-- <li><a href="reviews.html" class="nav-link" id="nav_reviews">시공후기</a></li> -->
```

이를 주석 기호(`<!--`와 `-->`)를 제거하여 아래와 같이 만듭니다.

```html
<li><a href="product-intro.html" class="nav-link" id="nav_prod_intro">제품소개</a></li>
<li><a href="reviews.html" class="nav-link" id="nav_reviews">시공후기</a></li>
```

### Step 2: 하단 푸터 메뉴 주석 풀기
마찬가지로 각 HTML 파일 코드 맨 아래쪽의 **푸터(Footer)** 영역으로 이동하여 아래 코드를 찾아 주석 기호를 지워줍니다.

```html
<!-- <li><a href="product-intro.html">제품소개</a></li> -->
<!-- <li><a href="reviews.html">시공후기</a></li> -->
```

이를 아래와 같이 활성화합니다.

```html
<li><a href="product-intro.html">제품소개</a></li>
<li><a href="reviews.html">시공후기</a></li>
```

## 🛒 3. 제품판매 페이지에 새로운 제품 추가하는 방법
`products.html` 파일에 새로운 제품 카드를 직접 추가하고 사진과 내용을 작성하는 방법입니다.

1. **템플릿 코드 복사**: `products.html` 파일 내부에서 `[제품 추가 템플릿 시작]` 주석을 찾습니다.
2. 바로 그 아래에 있는 `<!--` 로 시작하여 `-->` 로 끝나는 전체 블록 코드를 복제하여 붙여넣습니다.
3. **주석 해제**: 새로 붙여넣은 블록의 맨 앞 `<!--` 와 맨 뒤 `-->` 를 지워 코드를 활성화합니다.
4. **글 수정**: `<h3>추가할 새로운 제품 이름</h3>` 의 이름 부분과 `<p>제품 상세 설명...</p>` 의 내용을 고객이 보는 제품 소개 글로 고쳐 씁니다.
5. **스펙 수정**: 하단의 스펙 이름(`주요 스펙 항목 1`)과 스펙 값(`스펙 상세 내용 1`)을 알맞게 입력합니다.
6. **사진 연동**: 
   - 사용하고 싶은 실제 새로운 에어컨 사진(예: `my-ac.png`)을 `images/` 폴더 안에 복사해 넣습니다.
   - 코드 상의 `src="images/product-custom.png"` 부분을 `src="images/my-ac.png"` 로 직접 변경해 주시면 깔끔하게 새 사진이 연동됩니다.

---

## 💡 팁: 새로운 세부 페이지를 하나 더 만들고 싶을 때
기존의 다른 서브 페이지(예: `about.html` 또는 `product-intro.html`)를 복사(Ctrl + C)한 뒤 붙여넣기(Ctrl + V)하여 파일 이름만 새로 변경하세요.
그 후 헤더/푸터 네비게이션 메뉴에 해당 파일 경로를 링킹하면 손쉽게 무한대로 고급스러운 페이지를 추가하실 수 있습니다.
